## 可用示例

### 缓存日志记录演示 (`cache_logging_demo.py`)

演示如何使用 Redis 缓存机制的日志记录功能，包括：

- 缓存的基本操作（设置、获取、失效）
- 缓存降级场景（Redis 故障、缓存禁用）
- 结构化日志输出（JSON 格式）
- 日志分析方法和命令

**运行要求：**
- Redis 服务器运行在 `localhost:6379`（可选，示例会演示降级场景）
- 已安装项目依赖

**运行方法：**

```bash
# 激活 Conda 环境
conda activate mai_notebook

# 运行示例
python examples/cache_logging_demo.py
```

**预期输出：**

示例会输出三个场景的演示：

1. **场景 1：缓存启用 - 正常操作**
   - 展示缓存设置、命中、未命中、失效等操作的日志

2. **场景 2：缓存禁用 - 降级模式**
   - 展示缓存禁用时的降级行为

3. **场景 3：Redis 故障 - 自动降级**
   - 展示 Redis 连接失败时的自动降级

4. **日志分析示例**
   - 提供常用的日志分析命令
